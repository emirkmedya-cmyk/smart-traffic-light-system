# Smart Traffic Light Control System

## Project Description
This project implements a two-road smart traffic light system using a microcontroller. The system adjusts signal timing based on vehicle detection sensors and includes an LDR-based night mode for low-light conditions.

## Features
- Vehicle detection using IR sensors
- Adaptive traffic signal timing
- Finite State Machine implementation
- Night mode using LDR sensor

## Modules
1. Vehicle Detection Module
2. Traffic Signal Control Module
3. Timing and State Management Module
4. Adaptive and Night Mode Control Module

## Hardware Components
- Arduino Uno
- IR Sensors
- LEDs (Red, Yellow, Green)
- LDR Sensor
- Breadboard & Jumper Wires
